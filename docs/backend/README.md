# Obstor Backend
Obstor Backend adds S3 and [other supported protocol](../protocols/README.md) compatibility to third-party cloud storage providers.

## Supported Backends
- [NAS](nas.md)
- [Microsoft Azure Blob Storage](azure.md)
- [HDFS](hdfs.md)
- [S3](s3.md)
- [Google Cloud Storage](gcs.md)
- [SFTP](sftp.md)
